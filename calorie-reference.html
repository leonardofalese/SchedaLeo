Come funziona il parser

Ogni voce della scheda (es. "150g petto di pollo") viene analizzata in 3 passi:

PASSO 1 — Estrai quantità e unità
"150g petto di pollo" → qty = 150, unit = "g", food = "petto di pollo"
"2 scatolette tonno" → qty = 2, unit = "scatolette", food = "tonno"
"1 frutto" → qty = 1, unit = "pz", food = "frutto"
"500 kcal" → kcal esplicite → 500 (usate direttamente)

PASSO 2 — Converti unità in grammi
g → grammi diretti
kg → × 1000
pz → peso variabile (vedi tabella sotto)
fette → 35g / fetta di pane · 10g / biscotto o galletta
scatolette → 80g tonno sgocciolato · 120g altri
busta · pacco → 100g

PASSO 3 — Cerca nel DB (chiave più lunga = più specifica vince)
"pasta integrale" → match "pasta integrale" (340) ✓
"pasta" → match "pasta" (350) ✓
Se non trovato → fallback generico 150 kcal/100g

FORMULA FINALE
kcal = round( grammi_totali × kcal_per_100g / 100 )

"140g pasta integrale"

"250g macinato manzo magro"
Qty = 250g · Stopword "magro" rimossa dalla ricerca
Cerca "macinato manzo magro" (chiave lunga) → 178 kcal/100g
⚠ Prima trovava "macinato manzo" (250 kcal) — differenza: 180 kcal su 250g!
= 445 kcal
"1 frutto"
Unit = "pz" · Food = "frutto" → peso = 150g
Cerca "frutto" → 55 kcal/100g
= 83 kcal
"3 fette pane in cassetta"
Unit = "fette" · food = "pane in cassetta" → non è biscotto → 35g/fetta
3 × 35g = 105g · Cerca "pane in cassetta" → 268 kcal/100g
= 281 kcal
"500 kcal"
Kcal esplicite rilevate → usate direttamente senza parsing
= 500 kcal

Formula BMR · TDEE · Bilancio

BMR (Mifflin-St Jeor)
Uomo: 10×peso + 6.25×altezza − 5×età + 5
Donna: 10×peso + 6.25×altezza − 5×età − 161

peso in kg · altezza in cm · età in anni
TDEE (con palestra)
TDEE = BMR × moltiplicatore attività

0 giorni/sett → ×1.20 (sedentario)
1–2 giorni/sett → ×1.375
3–4 giorni/sett → ×1.55
5–6 giorni/sett → ×1.725
7 giorni/sett → ×1.90

+ bonus volume: ogni 10.000 kg/sett (serie×rip×kg) → +0.015, max +0.15
Proiezione obiettivo
DIMAGRIRE (formula calorica — 7700 kcal = 1 kg grasso)
deficit/die = scheda_kcal − TDEE (deve essere negativo)
giorni = (kg_da_perdere × 7700) / |deficit/die|
più palestra → TDEE alto → deficit grande → stima più corta ✓

MASSA (formula fisiologica — limite è ormonale, non calorico)
kg/mese = 0.25 (0 gg) · 0.40 (1–2 gg) · 0.60 (3 gg) · 0.75 (4 gg) · 0.90 (5+ gg)
+ bonus volume: +0.08/+0.10 se volume ≥ 5.000 / 15.000 kg/sett
mesi = kg_da_guadagnare / kg_mese
più palestra → tasso mensile alto → stima più corta ✓
⚠ Nota su crudo vs cotto: la scheda usa i pesi come scritti. Se scrivi "140g pasta" intendendo la pasta cotta, usa il valore "pasta cotta" (131 kcal/100g invece di 350). Per essere preciso, specifica "140g pasta cotta" oppure "140g pasta integrale cotta".
